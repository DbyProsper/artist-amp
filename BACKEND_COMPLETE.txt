🎯 HOW TO USE
───────────────────────────────────────────────────────────────────────────────

1. START THE BACKEND:

   Windows:
   $ cd backend && run.bat

   Mac/Linux:
   $ cd backend && ./run.sh

   Manual:
   $ cd backend
   $ python -m venv venv
   $ source venv/bin/activate
   $ pip install -r requirements.txt
   $ python -m uvicorn main:app --host 127.0.0.1 --port 8000

2. START THE FRONTEND:

   In a new terminal:
   $ npm run dev

3. TEST THE BACKEND:

   Interactive API: http://127.0.0.1:8000/docs
   Health Check: http://127.0.0.1:8000/health

4. USE IN FRONTEND:

   The frontend at OnlineStudioPage.tsx will now:
   - Generate lyrics without errors
   - Generate beautiful cover art instantly
   - Support all AI-powered music generation features
   - Handle errors gracefully with proper error messages
