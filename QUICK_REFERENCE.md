# Quick Reference - New AI Studio Features

## 🎵 Music from Audio Tab
**Purpose:** Transform uploaded audio with AI models

**How to use:**
1. Upload an audio file (max 50MB)
2. Select AI model (Gemini or MusicGen)
3. Describe the transformation you want
4. Click "Generate Music"
5. Download or preview the result
6. Auto-saves to your playlist

**Example prompts:**
- "Make this more upbeat with electronic elements"
- "Add reverb and make it more ambient"
- "Transform this into a dance track"

---

## 🖼️ Image from Upload Tab
**Purpose:** Generate images based on reference images

**How to use:**
1. Upload a reference image (max 10MB)
2. Select AI model (Gemini or MusicGen)
3. Describe changes you want
4. Click "Generate Image"
5. Download the generated image
6. Copy the URL if needed

**Example prompts:**
- "Make the colors more vibrant"
- "Change this to abstract style"
- "Add more details and texture"

---

## 🔊 Audio Enhancement Tab
**Purpose:** Improve audio quality with different effects

**Enhancement Types:**
- **Denoise**: Remove background noise
- **Enhance**: Improve overall clarity
- **Normalize**: Balance volume levels
- **Reverb**: Add space/room effect

**How to use:**
1. Upload audio file
2. Select enhancement type
3. Click "Enhance Audio"
4. Review the result
5. Auto-saves to playlist

---

## 💬 AI Chat Tab
**Purpose:** Get creative suggestions and production advice

**Features:**
- Model selection (Gemini, MusicGen, OpenAI)
- Conversation history automatically tracked
- Press Ctrl+Enter to send quickly
- Helpful for brainstorming

**Example questions:**
- "What makes a good pop song?"
- "How do I improve this beat description?"
- "What's a good way to describe electronic music?"

---

## 🎨 Model Selection Guide

### Gemini AI
- Best for: Lyrics, detailed descriptions, creative ideas
- Strengths: Natural language, context-aware
- Use when: You need intelligent suggestions

### MusicGen
- Best for: Audio generation, beat creation, music processing
- Strengths: Musical understanding, audio synthesis
- Use when: Generating or modifying audio

### OpenAI (Chat only)
- Best for: General questions, brainstorming
- Strengths: Broad knowledge, creative suggestions
- Use when: Discussing music theory or production

## 📋 File Upload Best Practices

### Audio Files
- **Format**: MP3, WAV, OGG, M4A
- **Max Size**: 50MB
- **Best Quality**: WAV or FLAC (lossless)
- **Examples**: Your recordings, stems, reference tracks

### Image Files
- **Format**: JPG, PNG, WebP, GIF
- **Max Size**: 10MB
- **Best Size**: At least 512x512 pixels
- **Examples**: Cover art, photos, concept art

## ⚡ Tips & Tricks

### Auto-Save Feature
- All generations automatically save to "AI Generated Music" playlist
- No manual save needed
- Find them later in your library
- Includes metadata (model used, generation type)

### Keyboard Shortcuts
- **Ctrl+Enter** in chat: Send message quickly
- **Tab to navigate**: Works in all input fields
- **Enter in textarea**: New line (use Ctrl+Enter to send)

### File Upload Tips
- Drag & drop files for faster upload
- Large files take longer to process
- Keep file formats consistent
- Test with different files to compare results

### Error Recovery
- Click the X on error messages to dismiss
- Loading will prevent accidental double-submissions
- Check file size/format if upload fails
- Try a smaller file if timeout occurs

✓ Check browser audio permissions
✓ Verify CORS settings
✓ Try different format

### Chat Not Responding
✓ Check message is not empty
